MaterialStateType¶

Process::Process::MaterialStateType

Overview¶

Concrete Class

This data structure represents the properties of a material or a single material phase. The MaterialState data structure for a Stream is built up of one or more of these data structures: one for the total properties and then one for each of the phases present.
