Process¶

Process::Process

Overview¶

Package

Members¶

Classes¶

Name

Type

Absorbing

Concrete Class

A subtype of SeparatingByPhysicalProcess where separation is based on mass transfer of material from a gas to a liquid. This type is also used to represent desorbing, where the mass transfer is from the liquid to the gas.

Adsorbing

Concrete Class

A subtype of SeparatingByPhysicalProcess where separation is based on mass transfer of material from a fluid to the surface of a solid. This type is also used to represent the reverse process, where the mass transfer is from the surface to the fluid.

Agglomerating

Concrete Class

A solids-processing process for IncreasingParticleSize in which particles bind together and coalesce into larger particles, held together by relatively weak forces.

Agitating

Concrete Class

This ProcessStepDetail provides shear forces to liquids in another process, with the purpose of mixing, increasing heat transfer rates or increasing mass transfer rates.

BlowingDown

Concrete Class

This SteeringFlow process is used to depressurise or blow down a process, removing the contents of a process rapidly and safely to prevent unsafe conditions or shut down the process. This process is usually realized as a set of blow-down or depressurisation valves.

Boiling

Concrete Class

This SupplyingThermalEnergy process vaporises a fluid. This process can be realized as a boiler, heat exchanger or fired heater. The process can also be used to represent the re-boiling subprocess in a distillation process.

CalculatingProcessVariable

Abstract Class

This InstrumentationActivity calculates an output that is some mathematical function of the inputs. This is a parent type for specialized calculations with specific purposes.

CalculatingRatio

Concrete Class

This CalculatingProcessVariable process calculates an output signal that is the ratio of the two input signals. It can be used to specify and design ratio control functions.

CalculatingSplitRange

Concrete Class

This CalculatingProcessVariable process calculates two output signals so that a single input signal can used to manipulate two control elements using a split range. It can be used to specify and design split range control functions.

Coalescing

Concrete Class

This is an IncreasingParticleSize process in which small droplets of a liquid phase are made to coalesce into larger droplets, which can then be separated from the mixture.

Composition

Concrete Class

This data item is an array of numbers that describe the chemical composition of a material. The order of items in the array is determined by the list of chemical components (ListOfMaterialComponents) in the corresponding MaterialTemplate object (see MaterialTemplate.ListOfComponents).

Compressing

Concrete Class

This GeneratingFlow process represents compression of gases to increase pressure and generate a flow of gas. The physical principle used to compress the gas can be specified by selecting a value from the CompressionMethod enumeration.

ContactingInPacking

Concrete Class

This ProcessStepDetail is used to represent a process carried out in a packed bed. Its role is to represent packing shown in process flow diagrams and to support the functional breakdown of complex processing systems. See also ContactingOnTray.

ContactingOnTray

Concrete Class

This ProcessStepDetail is used to represent a process carried out in an assembly of trays. Its role is to represent trays shown in process flow diagrams and to support the functional breakdown of complex processing systems. See also ContactingInPacking.

ControllingProcessVariable

Concrete Class

This InstrumentationActivity represents a process control activity. A block of this type calculates the value of a manipulated variable so that a measured variable is held at a specified set point. The set point can be supplied as an external InformationFlow.

ConveyingSignal

Concrete Class

This process transmits a signal between two processes. In practices, this process is only used if there will be a need to invest in specialized equipment and facilities, or if the properties of the signal are different at the input and output of the process. Most signal connections are represented by InformationFlow objects between InformationPort instances.

Cooling

Concrete Class

This RemovingThermalEnergy process has the purpose of reducing the enthalpy content of a material through heat exchange with the environment or a cooling medium. See also ExchangingThermalEnergy, which can be used to model the same process in cases where the cooling medium system is defined.

Crushing

Concrete Class

This ReducingParticleSize process takes solid material and breaks it into smaller pieces by an externally applied mechanical force. See also Grinding and Milling.

Crystallizing

Concrete Class

This IncreasingParticleSize process is based on the formation of crystals from a feed fluid. This then allows the separation of the resulting solids from the liquid.

CustomMaterialComponent

Concrete Class

A CustomMaterialComponent defines a custom chemical compound used to specify the chemical composition of a material. These compounds are usually pseudocomponents that are used to simulate the thermodynamic properties of complex mixtures of organic compounds with high molecular weight.

Cutting

Concrete Class

This ReducingParticleSize process separates solid material into smaller parts using an edged implement or knife.

Distilling

Concrete Class

This SeparatingByThermalProcess process uses boiling, condensing and countercurrent contacting of liquid and vapour to separate chemical components with a difference in relative volatility.

Draining

Concrete Class

This supporting SteeringFlow process is used to manage leakages and handle the emptying of a plant for maintenance.

DrivingByEngine

Concrete Class

This SupplyingMechanicalEnergy process uses an internal combustion engine as its source of energy.

DrivingByMotor

Concrete Class

This SupplyingMechanicalEnergy process uses an electric motor as its source of energy.

DrivingByTurbine

Concrete Class

This SupplyingMechanicalEnergy process uses an turbine as its source of energy.

Drying

Concrete Class

This SeparatingByThermalProcess process removes moisture or liquids from a solid material, producing a solid product with a lower content of the liquid components.

ElectricalEnergyFlow

Concrete Class

A ProcessConnection that defines the flow of energy between two instances of ElectricalEnergyPort. This is used to model flows of electricity into out of the port. The form of electrical energy is defined by the properties of the object.

ElectricalEnergyPort

Concrete Class

An EnergyPort that models the exchange of electrical energy between ProcessStep objects. The form of electrical energy exchanged is defined by the values of the properties of the ElectricalEnergyFlow into or out of the port.

Emitting

Concrete Class

This type represents the destination for a flow out of a process that can be regarded as an emission to the environment. It can be used instead of a Sink. It is provided as a convenience for calculating emissions and sustainability calculations.

EnergyFlow

Concrete Class

A ProcessConnection that defines the flow of energy between two instances of EnergyPort. This is used to model flows of energy where the form of energy is unspecified. This type usually abstract and child types of this type are used to model energy flows with a specified form of energy.

EnergyPort

Concrete Class

A Port that models the transfer of energy, of whatever form, into our out of a ProcessStep object. This type is parent to specific port types that specify the form of energy exchanged. This type can be used to represent energy requirements where the type of energy is unspecified. In practice, this type is abstract, and the child types are used in modelling.

Evaporating

Concrete Class

This SeparatingByThermalProcess process applies thermal energy to drive off moisture or other volatile liquid as gas. This results in a vapour product and a more concentrated liquid or solid product.

ExchangingThermalEnergy

Concrete Class

This process transfers thermal energy between two or more streams of material. This process is realized by a heat exchanger.

Extruding

Concrete Class

This FormingSolidMaterial process takes solid material and forms it by forcing it through a specially designed opening, so that the material is given a specific shape.

FeedingMaterial

Concrete Class

This SteeringFlow process supplies material to a process at a controllable rate. This process can be used to represent solid feeding processes or dosing systems.

Filtering

Concrete Class

This SeparatingMechanically process separates solids from fluids by passing the fluid through a porous barrier.

Flaring

Concrete Class

This SupplyingThermalEnergy process represents the flare system in a process plant.

Flocculating

Concrete Class

This IncreasingParticleSize process produces larger particles, flocs, through aggregation and coalesence under the influence addition of surfactant chemicals.

FormingSolidMaterial

Concrete Class

This process takes a material containing solids and forms the solid material into particles with specified sizes and shape. It is the parent type for Pelletizing and Extruding.

GeneratingACPower

Concrete Class

This SupplyingElectricalEnergy process produces alternating current electrical energy from mechanical energy.

GeneratingDCPower

Concrete Class

This SupplyingElectricalEnergy process produces alternating current electrical energy from mechanical energy.

GeneratingFlow

Concrete Class

This is a base type for processes with the purpose of generating flow and/or increasing pressure of a fluid stream. The child types are Pumping and Compressing.

GeneratingInFuelCell

Concrete Class

This process generates electricity using a fuel cell.

GeneratingSteam

Concrete Class

This process produces steam for use in other processes.

Grinding

Concrete Class

This ReducingParticleSize process produces fine particles by abrasion of solid material.

HeatingElectrical

Concrete Class

This SupplyingThermalEnergy process increases the enthalpy content of one or more material streams through the application of electrical energy.

HeatingInFurnace

Concrete Class

This SupplyingThermalEnergy process increases the enthalpy content of one or more material streams by passing them through a fired heater or furnace.

Humidifying

Concrete Class

This Mixing process increases the water or liquid content of a material stream by mixing a feed with the water or liquid.

IncreasingParticleSize

Concrete Class

This process takes a material containing solids and performs an operation so that the solids have a larger particle size.

InformationFlow

Concrete Class

A ProcessConnection that defines the flow of information between two instances of InformationPort. This is used to model signals and other information flows into out of the port.

InformationPort

Concrete Class

A Port that models the transfer of information, of whatever form, into our out of a ProcessStep object. This type is used to represent control signals.

InformationVariant

Concrete Class

This data structure is embedded in the InformationFlow object and can be configured to represent scalars and arrays of boolean, integer and real variables.

InstrumentationActivity

Abstract Class

This is the base type for control and instrumentation functions that are specified as part of the process design.

InstrumentationSystemActivity

Concrete Class

An InstrumentationSystemActivity is a collection of related InstrumentationActivity instances that together perform a specified function. Thus a MeasuringProcessVariable and a ControllingProcessVariable activity can be grouped into a single InstrumentationSystemActivity that represents the control loop.

Kneading

Concrete Class

This Mixing process produces a solid material as a uniform mass by mechanically folding, pressing and stretching of the material.

LimitingFlow

Concrete Class

This SteeringFlow process is used to ensure that the flow of material remains below a specified value. This process is usually realized using a flow orifice.

ListOfMaterialComponents

Concrete Class

This is an ordered list of PureMaterialComponent and CustomMaterialComponent objects. This is used in the MaterialTemplate to determine the structure of composition vectors in the Stream and MaterialStateType objects.

MaterialComponent

Abstract Class

This is the base type to chemical components in the model. It is parent class for PureMaterialComponent and CustomMaterialComponent.

MaterialPort

Concrete Class

A Port that models the transfer of material, of whatever form, into our out of a ProcessStep object. This type is used flow of material: solids, liquids and gases.

MaterialState

Concrete Class

This data structure contains the properties for a Stream. It contains one or more MaterialState objects: one for the total properties and then one for each of the phases present.

MaterialStateType

Concrete Class

This data structure represents the properties of a material or a single material phase. The MaterialState data structure for a Stream is built up of one or more of these data structures: one for the total properties and then one for each of the phases present.

MaterialTemplate

Concrete Class

This data structure defines the structure of Stream objects that correspond to a specific type of material. This structure contains information about the thermodynamic model used to represent this type of material. It contains the ListOfMaterialComponents: pure and pseudo components used to represent the system. It also determines the legend for a stream table when the model is used to represent a Process Flow Diagram.

MeasuringProcessVariable

Concrete Class

This InstrumentationActivity measures a process variable and supplies its value at an outlet InformationPort. The measured variable is identified by reference to a parameter in any process step or port.

MechanicalEnergyFlow

Concrete Class

A ProcessConnection that defines the flow of mechanical energy two instances of MechanicalEnergyPort.

MechanicalEnergyPort

Concrete Class

A Port that models the transfer of mechanical energy into or out of a ProcessStep object.

Milling

Concrete Class

This ReducingParticleSize process divides solid material into smaller particles with precise sizes and shapes.

Mixing

Concrete Class

This is the base class for processes that combine several flows of material into a single flow.

MixingSimple

Concrete Class

This is a Mixing process that simply combines one or more inlet flows into an outlet flow. This process step is used to represent converging branches of flows in a PFD or BFD. It is the counterpart of Splitting.

Packaging

Concrete Class

This process takes a flow of material and processes it into packages that allow transport and distribution.

Pelletizing

Concrete Class

This FormingSolidMaterial process takes solid material and forms it into, pellets, small particles of uniform size and shape.

Port

Abstract Class

A port represents a transfer of material, energy or information into or out of a process step. It provides the anchor point for connecting two process steps to each other. The term is derived from SysML and IDEF0 representations of processes.

PreventingBackflow

Concrete Class

This SteeringFlow process is used to ensure that a flow of material cannot be reversed. This process is used in safety design and is realized by a check valve or non-return valve.

ProcessConnection

Abstract Class

This data structure is the abstract parent for all connections between blocks in the DEXPI Process model.

ProcessStep

Abstract Class

The ProcessStep type is the parent type for all process steps in the model.

ProcessStepDetail

Abstract Class

This type is the parent type for processes that cannot be run independently of another process.

Pumping

Concrete Class

This GeneratingFlow process represents transfer of mechanical energy to a liquid to increase pressure and generate a flow. The physical principle used to compress the gas can be specified by selecting a value from the PumpingMethod enumeration.

PureMaterialComponent

Concrete Class

A PureMaterialComponent represents a MaterialComponent that is modelled as a pure chemical compound with defined composition and structure. This is used in the MaterialTemplate to document the composition vector and define properties for use in the process.

ReactingChemicals

Concrete Class

This process performs chemical reactions on material in order to produce products of value or to remove undesirable chemical compounds.

ReducingParticleSize

Concrete Class

A process where a solid material is broken down into particles with a smaller size. This is the parent class for Cutting, Milling, Crushing and Grinding.

RegulatingFlow

Concrete Class

A SteeringFlow process where a flow of material is manipulated over a continuous range of values. The desired flow rate is supplied in the XG1 InformationPort.

RelievingOverpressure

Concrete Class

A SteeringFlow process where a flow of material is manipulated to ensure that the pressure in a system shall not exceed a specified value.

RelievingVacuum

Concrete Class

A SteeringFlow process where a flow of material is manipulated to ensure that the pressure in a system shall not become sub-atmospheric.

RelievingVacuumAndOverpressure

Concrete Class

A SteeringFlow process where a flow of material is manipulated to ensure that the pressure in a system shall neither become sub-atmospheric nor exceed a specified value.

RemovingThermalEnergy

Concrete Class

This process removes enthalpy from a material in order to reduce temperature or change the state of the material. See also Cooling.

RotaryMixing

Concrete Class

This Mixing process uses an externally powered rotating impeller to produce a homogeneous material.

Separating

Concrete Class

This is the base type for all processes where one or more feeds are transformed into streams of material, each with different intensive properties and chemical composition.

SeparatingByCentrifugalForce

Concrete Class

This is a SeparatingByPhaseSeparation process where a fluid mixture is separated into a heavy phase and light phase through the external application mechanical energy to produce centrifugal forces.

SeparatingByContact

Concrete Class

This is a SeparatingByPhysicalProcess process where materials are brought into contact with each to transfer mass and energy.

SeparatingByCyclonicMotion

Concrete Class

This is a SeparatingByPhaseSeparation process where a fluid mixture is separated into a heavy phase and light phase through the centrifugal forces generated by the flow of fluids through a cyclone arrangement.

SeparatingByElectromagneticForce

Concrete Class

This is a Separating process where a material mixture is separated by the application of an electrical or magnetic field. This type is parent to SeparatingByElectrostaticForce and SeparatingByMagneticForce.

SeparatingByElectrostaticForce

Concrete Class

This is a SeparatingByElectromagneticForce process where a material mixture is separated by the application of an electrostatic field.

SeparatingByFlash

Concrete Class

This is a SeparatingByThermalProcess process where a fluid mixture is separated into a vapour phase and a liquid phase in a vessel, usually after a sudden decrease in pressure. This is a single-stage process, unlike Distilling.

SeparatingByGravity

Concrete Class

This SeparatingByPhaseSeparation process separates substances based on the influence of gravity on substances with different density or specific gravity.

SeparatingByIonExchange

Concrete Class

This SeparatingByPhysicalProcess is a reversible interchange of one kind of ion present on an insoluble solid with another of like charge present in a solution surrounding the solid with the reaction being used especially for softening or demineralizing water, the purification of chemicals, or the separation of substances.

SeparatingByMagneticForce

Concrete Class

This is a SeparatingByElectromagneticForce process where a material mixture is separated by the application of a magnetic field.

SeparatingByPhaseSeparation

Concrete Class

This is a Separating process where the materials form distinct phases, which can then be separated.

SeparatingByPhysicalProcess

Concrete Class

This Separating process uses a physical phenomenon, such as mass transfer, ion exchange or surface tension to separate materials. Its child types are SeparatingByContact, SeparatingByIonExchange, Absorbing, Adsorbing and SeparatingBySurfaceTension.

SeparatingBySurfaceTension

Concrete Class

This is a Separating process where the materials are separated through a difference in surface forces between different liquid phases or between liquids and solid particles. Froth flotation is a typical process of this type.

SeparatingByThermalProcess

Concrete Class

This is a Separating process where the materials are separated through the application of heat or thermal energy.

SeparatingMechanically

Concrete Class

This is a parent process where the materials are separated by a mechanical operation, such as Filtering, Skimming or Sieving.

ShuttingOffFlow

Concrete Class

This is SteeringFlow process with the purpose of shutting off flow for operational and/or safety reasons. It is usually realized by an on-off valve.

Sieving

Concrete Class

This SeparatingMechanically processs for solids is based on particle size. Material is passed over a series of screens with a specified aperture size. Smaller particles pass through the screens, while larger particles remain on the screen.

Sink

Concrete Class

This type is used to represent a flow of material out of the process. It can be used to represent an outgoing off-page connector in a PFD or BFD.

Skimming

Concrete Class

This is a SeparatingMechanically process in which a lighter immiscible fluid, such as oil, is separated from a heaver fluid, such as water, by mechanical wiping of the surface of the heavier fluid.

Source

Concrete Class

This type is used to represent a flow of material into the process. It can be used to represent an incoming off-page connector in a PFD or BFD.

Splitting

Abstract Class

This type is used to represent the splitting of a material or energy flow. This type is abstract, and the child types SplittingMaterial and SplittingEnergy are used in practice.

SplittingEnergy

Concrete Class

This type is used to represent the splitting of an energy flow. The intensive properties of the flow remain the same, but the total amount of flow (power) is divided among the outlet ports.

SplittingMaterial

Concrete Class

This process splits an incoming flow of material into two or more outgoing flows of material. The intensive properties of the material flows remain the same.

StabilizingDistilling

Concrete Class

This Distilling process has the purpose of producing a stabilized liquid product by removing the volatile chemical compounds that would otherwise make the product unstable for transport or storage.

StaticMixing

Concrete Class

This Mixing process uses a stationary element in the flow of material to produce the necessary homogeneity in the material that comes out of the process. It is realized by a StaticMixer.

SteeringFlow

Abstract Class

The base type for processes that manipulate the rate of flow of material into and out of the process step. This process is usually realized using a valve. Child types of this type have been specified to support design of the process control and safety systems.

StoringElectricalEnergy

Concrete Class

This process step stores electrical energy. It can be connected to a charging flow of electricity and can also be used as a supplier of electrical energy to other process steps. The principle of storage is unspecified.

StoringEnergy

Abstract Class

This process step stores electrical energy.

StoringFluids

Concrete Class

This StoringMaterial process accumulates fluids: gases and liquids. It can be connected to a filling flow of material and an optional emptying flow of material. In this way it can be used as supplier of material for other process steps. The principle and construction of the storage is unspecified. The sub-types of this type are differentiated according to whether the storage is pressurised or not. StoringInTank is used for atmospheric storage of liquids, whereas StoringInPressureVessel is used for storage of gases and fluids at above-atmospheric or sub-atmospheric pressure.

StoringInBattery

Concrete Class

This StoringElectricalEnergy process step stores electrical energy in a battery. It can be connected to a charging flow of electricity and can also be used as a supplier of electrical energy to other process steps.

StoringInPressureVessel

Concrete Class

This StoringFluids process accumulates fluid materials in a PressureVessel. This process assumes atmospheric storage of the material at pressures above or below atmospheric pressure. Use StoringInTank where fluid is stored at atmospheric conditions. It can be connected to a filling flow of material and an optional emptying flow of material. In this way it can be used as supplier of material for other process steps.

StoringInSilo

Concrete Class

This StoringSolids process is realized by a Silo.

StoringInTank

Concrete Class

This StoringFluids process accumulates fluid, solid or mixed materials in a Tank. This process assumes atmospheric storage of the material. Use StoringInPressureVessel where fluid or gas is to be stored at above-atmospheric or sub-atmospheric conditions. It can be connected to a filling flow of material and an optional emptying flow of material. In this way it can be used as supplier of material for other process steps.

StoringMaterial

Abstract Class

An abstract process step that is the parent class for processes that accumulate material. It is often the intention of these steps that the accumulated material can be used as supply for other process steps.

StoringSolids

Concrete Class

This StoringMaterial process accumulates solid material. It can be connected to a filling flow of material and an optional emptying flow of material. In this way it can be used as supplier of material for other process steps. The principle and construction of the storage is unspecified.

StoringThermalEnergy

Concrete Class

This process accumulates and stores thermal energy for use in other processes. A calorifier is an example of this process in HVAC systems.

Stream

Concrete Class

This data structure defines the properties of a flow of fluid or solid material between two MaterialPort objects.

StrippingDistilling

Concrete Class

This Distilling process has the purpose of removing volatile components from a fluid.

SupplyingElectricalEnergy

Concrete Class

This process is a source of electrical energy for use in other processes.

SupplyingFluids

Concrete Class

This process is a source of fluid materials (liquids, gases or multiphase mixtures) for use in other processes.

SupplyingMechanicalEnergy

Concrete Class

This process is a source of mechanical energy for use in other processes. It is the parent type for the DrivingByEngine, DrivingByMotor and DrivingByTurbine types.

SupplyingSolids

Concrete Class

This process is a source of solid material for use in other processes.

SupplyingThermalEnergy

Concrete Class

This process is a source of thermal energy for use in other processes.

SupplyingThermalEnergyWithBurner

Concrete Class

This ProcessStepDetail is used to specify a combustion process as a sub-process of some other process, such as a sub-type of SupplyingThermalEnergy. This process is usually realized by one or more Burner components.

ThermalEnergyFlow

Concrete Class

This data structure defines the properties of a flow of heat or thermal energy between two ThermalEnergyPort objects.

ThermalEnergyPort

Concrete Class

This is a Port that exchanges heat or thermal energy between processes.

TransformingProcessVariable

Concrete Class

This process performs a specified mathematical transformation on one or more input variables to calculate one or more output variables. The input and output variables are InformationPort objects owned by the type.

TransportingElectricalEnergy

Concrete Class

This process transmits electrical energy between processes.

TransportingFluids

Concrete Class

This process moves fluids: gas, liquid or multiphase flows, between processes. It is also used to model distribution processes, such as steam supply systems, gas reticulation networks, cooling water systems and flare systems. Three child types are used to specify the realisation of this process, as the realization has implications for process design. The child types are TransportingFluidsInChannel, TransportingFluidsInHose and TransportingFluidsInPipe.

TransportingFluidsInChannel

Concrete Class

This TransportingFluids process is realized by an open channel or drain. This type can be used to model open drain systems.

TransportingFluidsInHose

Concrete Class

This TransportingFluids process is realized by a flexible connector or hose. This type can be used to model hose or flexible piping systems.

TransportingFluidsInPipe

Concrete Class

This TransportingFluids process is realized by a rigid connector, duct or pipe. This type can be used to pipeline systems, piping-based fluid distribution systems and ducting systems.

TransportingSolids

Concrete Class

This process moves solid material between processes. Two child types are used to specify the realisation of this process, as the realization has implications for process design. The child types are TransportingSolidsContinuously and TransportingSolidsDiscontinuously.

TransportingSolidsContinuously

Concrete Class

This process moves solid material between processes using a continuous operation, such as a conveyor of various types.

TransportingSolidsDiscontinuously

Concrete Class

This process moves solid material between processes using a discontinuous or batch operation, such as a vehicle, container or crane.

VacuumDistilling

Concrete Class

This is a Distilling process that is conducted at sub-atmospheric pressures.
