Cooling¶

Process::Process::Cooling

Overview¶

Concrete Class

This RemovingThermalEnergy process has the purpose of reducing the enthalpy content of a material through heat exchange with the environment or a cooling medium. See also ExchangingThermalEnergy, which can be used to model the same process in cases where the cooling medium system is defined.
